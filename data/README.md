# Data Folder

This folder contains dataset references and processed artifacts used for the fraud detection pipeline.

## Dataset

The project uses the popular Credit Card Fraud Detection dataset from Kaggle.

Dataset source:
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

## Structure

* `raw/` → original dataset files (not committed to GitHub)
* `processed/` → trained model artifacts and preprocessing objects
