#  Fraud Detection System (ML + FastAPI + Streamlit)

An end-to-end machine learning system to detect fraudulent transactions in real-time using advanced models, explainability techniques (SHAP), and a production-style API + dashboard.

---

## Highlights

-  Fraud detection using ML models (XGBoost / LightGBM)
-  FastAPI backend for real-time inference
-  Streamlit dashboard for interactive analysis
-  SHAP explainability (why a transaction is fraud)
-  Batch + single prediction support
-  Clean ML pipeline (EDA → Features → Model → API → UI)
